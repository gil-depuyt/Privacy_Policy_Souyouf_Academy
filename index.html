Règles de Confidentialité - Souyouf Brussels Academy

Chez Souyouf Brussels Academy, nous respectons votre vie privée et nous nous engageons à protéger les données personnelles que vous nous confiez. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations dans le cadre de notre application.

1. Informations que nous collectons

Souyouf Brussels Academy peut collecter les types de données suivants :

  • Données personnelles : Si vous interagissez avec notre application, nous pouvons collecter des informations comme votre nom, adresse email, etc., mais seulement si vous nous les fournissez volontairement (par exemple, via un formulaire de contact).
  • Données d'utilisation : Nous collectons des données anonymes concernant la manière dont vous utilisez l'application (ex : nombre de visites, fonctionnalités utilisées).

2. Utilisation des données collectées

Nous utilisons les données collectées pour les finalités suivantes :

  • Amélioration de l'expérience utilisateur.
  • Analyse des performances de l'application.
  • Communication avec les utilisateurs si nécessaire.

3. Partage des données avec des tiers

Nous ne partageons pas vos données personnelles avec des tiers, sauf si cela est nécessaire pour le bon fonctionnement de notre application, comme l'utilisation de services tiers tels que Google Analytics ou Firebase, qui peuvent collecter des informations anonymes à des fins d'analyse.

4. Sécurité des données

Nous prenons des mesures techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, perte ou divulgation.

5. Vos droits

Vous avez le droit de :

  • Accéder aux informations que nous détenons sur vous.
  • Demander la correction de vos informations.
  • Demander la suppression de vos informations.

6. Modifications des règles de confidentialité

Nous nous réservons le droit de mettre à jour cette politique de confidentialité. Toute modification sera publiée sur cette page, et la date de la mise à jour sera indiquée en bas du document.

7. Contact

Si vous avez des questions concernant notre politique de confidentialité, vous pouvez nous contacter à l'adresse suivante : souyoufinfo@gmail.com.
